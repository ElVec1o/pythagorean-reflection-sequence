----- NAME -----
Number of distinct affine isometries at relaxed word-length n in the side-reflection group of a right triangle in the plane with positive unequal rational legs (the relaxed companion of A396406).

----- DATA -----
1, 3, 5, 8, 13, 21, 34, 55, 91, 148, 235, 371, 590, 931, 1451, 2254, 3513, 5455, 8418, 12959, 19949, 30640, 46905, 71699, 109490, 166969, 254047, 386192, 586349, 889599, 1347444, 2039911, 3084135

----- OFFSET -----
0,2

----- COMMENTS -----
Let T be a right triangle in Q^2 with positive unequal legs and let W be the group generated by the reflections R_0, R_1, R_2 in its three sides (the same group as in A396406). Each element g of W is realized by walks in a transition system. The true word length ell_T(g) is the least length over realizations forming a single open walk; the relaxed length ell_R(g) is the least length over all realizations, closed walks ("isolated cycles") being permitted at no extra cost beyond their own crossings. These satisfy ell_T(g) = ell_R(g) + 2*c(g), where c(g) >= 0 is the least number of isolated cycles in a relaxed-optimal realization. A396406 counts elements by ell_T; a(n) here counts them by ell_R.

Since ell_R <= ell_T, this sequence dominates A396406 termwise: a(n) >= A396406(n), with equality for 0 <= n <= 7 and first strict inequality at n = 8 (a(8) = 91 while A396406(8) = 89). Both have the same growth rate: a(n+1)/a(n) -> beta_2 = 1.4916177871..., about 1.49 at the computed depths.

Conjecturally independent of the triangle T for pairwise distinct rational legs, as for A396406. Computed from the catalytic generating function for the relaxed length (Bonfioli 2026), cross-checked against exact-rational breadth-first search on the (1,2) triangle.

The building blocks of V are transcendental over Q(q) unconditionally (their power series has the unit circle as a natural boundary, by the Polya-Carlson theorem); V itself is transcendental over Q(x) (Bonfioli 2026).

----- FORMULA -----
a(n+1)/a(n) -> beta_2 = 1.4916177871... (the same growth rate as A396406); the exact value and the algebraic nature of the generating function are open.

a(n) >= A396406(n) for all n, with equality if and only if n <= 7.

No linear recurrence over Q of order <= 19 fits the data (as for A396406).

----- EXAMPLE -----
a(n) = A396406(n) for 0 <= n <= 7; the relaxed and true counts first differ at n = 8, where a(8) = 91 while A396406(8) = 89 (two elements of true length 8 have a strictly shorter relaxed realization). For 1 <= n <= 7 both equal A000045(n+3): 3, 5, 8, 13, 21, 34, 55.
