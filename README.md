# The Universal Right-Triangle Reflection Sequence

[![DOI](https://zenodo.org/badge/1235839920.svg)](https://doi.org/10.5281/zenodo.20370090)
[![OEIS](https://img.shields.io/badge/OEIS-A396406-blue)](https://oeis.org/A396406)
[![Lean](https://img.shields.io/badge/Lean%204-checked-success)](./lean/)

A structural rigidity theorem in the geometry of right-triangle reflection groups:
for every right triangle in the plane with **positive unequal** rational legs, the
BFS orbit-growth sequence of its three side-reflections is the *same* sequence
$u_d$, independent of the triangle.

**Sequence** (first 39 terms, offset 0):

```
1, 3, 5, 8, 13, 21, 34, 55, 89, 144, 225, 351, 554, 875, 1345, 2066,
3203, 4971, 7574, 11543, 17683, 27108, 41067, 62263, 94622, 143881,
217101, 327832, 495443, 749195, 1127236, 1697179, 2554961, 3848384,
5777651, 8679441, 13031206, 19574659, 29338781
```

For $1 \le n \le 9$, $u_n = F(n+3)$ — the Fibonacci numbers. The first
deviation from Fibonacci occurs at $n=10$: $u_{10} = 225 = F(13) - 8$,
with the deficit of exactly 8 accounted for by 8 explicit length-10
affine relations, symbolically verified in $\mathbb{Q}(a, b)$.

For the **isosceles** case ($a = b$) the sequence is genuinely different
(extra leg-swap stabilizer): $1, 3, 5, 8, 11, 13, 16, \ldots$ from depth 4
onward. The universality theorem applies only to the unequal-leg regime.

## Main results

1. **Universality Theorem.** For every right triangle $T$ with positive
   unequal rational legs, $\ker(\rho_T)$ is the cyclic left ideal
   $\mathbb{Z}[Q] \cdot h$ generated by $h = (c-1)(Y-1)$, where $c$ is the
   half-turn around the right-angle vertex (central in the linear
   quotient $Q$). Hence $u_d$ is independent of $T$.

2. **Closed-form $n$-dimensional growth rate.** The right-corner
   orthoscheme reflection group in $\mathbb{R}^n$ with pairwise distinct
   legs has asymptotic ratio
   $$r_n = 1 + 2\cos\left(\frac{2\pi}{n+3}\right).$$

3. **Class C unconditional faithfulness.** For pairwise distinct legs
   in every dimension $n \ge 3$, the affine representation is faithful,
   via the Schur-complement identity $\det Q_n = -\prod a_i^2$ and
   Mordell descent on Cremona elliptic curves 24a1 and 72a2.

## How to reproduce

Requires Python 3.10 or newer. No external Python dependencies for the
basic verification. The Rust components require `rustc` 1.70 or newer.

```bash
# Universality certificate — ~3 minutes on commodity hardware
python3 reproduce/universality.py --depth 25

# Smoke test (about 30 seconds)
python3 reproduce/universality.py --depth 15

# Full sequence to depth 42 (Rust, ~15 minutes, ~5 GB RAM)
cd code/rust_bfs && cargo build --release
./target/release/bonfioli_bfs --depth 42
```
